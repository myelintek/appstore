# Prerequistes
Install Node.js

Node.js TLS (v14.x):
```
Using Ubuntu
apt-get install -y ca-certificates
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

node -v: v14.18.0

npm -v: 7:24.2

# Installation

```
git clone git@github.com:myelintek/appstore.git
cd apptstore
npm install
npm start
```

# Build
```
npm run make
```
